# Test Scripts

This directory contains test scripts for the RI application.

## Available Scripts

### `test-notification.sh`
Tests the notification system functionality.

**Usage:**
```bash
./test/scripts/test-notification.sh
```

### `test-session-name-sync.sh`
Tests session name synchronization between frontend and backend.

**Usage:**
```bash
./test/scripts/test-session-name-sync.sh
```

## Running Tests

All test scripts can be executed from the project root directory:

```bash
# Run notification tests
./test/scripts/test-notification.sh

# Run session name sync tests
./test/scripts/test-session-name-sync.sh
```

## Notes

- Test scripts require the application to be running
- Some tests may require specific configuration settings
- Check individual script files for more detailed documentation
